# easy-cron

#### Description
这是基于Vue.js和iviewui封装一个crontab表达式的组件，可以方便、快捷直观地定义cron表达式，并支持给定开始时间后的执行时间点，方便排除错误。本工程包括测试代码，和vue原始文件。
本组件实现参考以下两位大佬：
@远望の无限：https://gitee.com/ywwxhz/CronExpGenerator
@ldy：https://gitee.com/lindeyi/vue-cron
